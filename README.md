หน้าแรกอยากให้เป็นภาพของฉันอยู่ทางด้านขวา ด้านซ้ายอยากให้เป็นคำว่า PORTFOLIO ตัวหนา จะเป็นฟิลขาวแล้วไล่ไปใสๆ ฟิลไอโอเอส ข้างล่างจะมีภาษาไทย พิมพ์ว่า "แฟ้มสะสมผลงาน" และข้างล่างจะเป็นชื่อของฉัน "นายพุฒิพงศ์ พุ่มเจริญ" ข้างล่างจะเป็นชื่อโรงเรียน " โรงเรียนกระทุ่มแบน“วิเศษสมุทคุณ” "   ซึ่งในตัวแถบด้านล่างจะเป็นบล็อคให้เลือกหน้าต่างๆ สามารถกดเข้าได้ทุกอัน
หน้าต่างแรก หัวข้อ "Statement of Purpose (SOP)"
ในหน้านี้จะขึ้นแสดงเอกสารของฉัน สามารถกดต่อไปยังหน้าถัดไป
หน้าต่างที่ 2 หัวข้อ "Who I am"
